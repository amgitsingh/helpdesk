# Project Scope

## AI Powered Ticket Management System

### Problem
    We receive hundreds of support email daily, our agents manually read and respond to each ticket- which is slow and leads to impersonal canned responses.
    
### Solution
    We will build an AI powered ticket management system that will automatically read and respond to each ticket- delivering fast and more pesonalised responses to students while freeing up the agents for complex issues.

## Features
    - Receive support email and create tickets
    - Auto-generate human friendly responses using knowledge base
    - Ticket list with filtering and sorting
    - Ticket detail view
    - AI-powered ticket classification
    - AI summaries
    - AI suggested replies
    - User management (admin only)
    - Dashboard to view and manage all tickets

## Ticket Statuses
    - **Open** – newly created, awaiting action
    - **Resolved** – a response has been sent or issue addressed
    - **Closed** – ticket is finalised and no further action needed

## Ticket Categories
    - General Question
    - Technical Question
    - Refund Request

## Roles & Access
    - **Admin** – the system is seeded with a single admin on deployment. The admin can create and manage agent accounts.
    - **Agent** – can view, respond to, and manage tickets
